Binary

Converting to binary

Consider the number $19$. If we want to convert it to binary, we want to find the largest power of two ($2^x$). In this case, that is $2^4 = 16$. We can use this to construct a table of values for all of the powers of 2 from $2^4$ to $2^0$.

Power of 2 ($x$) 4 3 2 1 0
$2^x$ 16 8 4 2 1
Fits in

From here, we can see that we can remove $16$ (leaving $3$), which we will note as a $1$ in the first column of the last row. We can repeat this process. Neither $8$ nor $4$ fit in $3$, so write $0$ in their columns. $2$ and $1$ fits in, so we write $1$ in their columns. Now we have this table:

Power of 2 ($x$) 4 3 2 1 0
$2^x$ 16 8 4 2 1
Fits in 1 0 0 1 1

If we read that from left to right, we will get the binary representation, $10011$.

Converting from binary

Converting from binary back to decimal is significantly simpler. Consider the table we created before:

Power of 2 ($x$) 4 3 2 1 0
$2^x$ 16 8 4 2 1
Fits in 1 0 0 1 1

The bottom number in Fits in depicts the number of times the base ($2^x$) can fit into the number. To convert it back, we can simply multiply each number by the base, and add them all together.

$$16 \times 1 + 8 \times 0 + 4 \times 0 + 2 \times 1 + 1 \times 1 = 19$$

Octal

Converting to octal

The process of converting to octal is very similar to converting to binary. Again, we want to find the largest number that matches $8^x$. Taking the examples from before, we can see that $8^1 = 8$ is the largest number that fits in $19$. We can construct a table of values for all of the powers of 8 from $8^1$ to $8^0$.

Power of 8 ($x$) 1 0
$8^x$ 8 1
Fits in

Note that the largest multiple of $8$ that fits into $19$ is $16$. So, we need to note the multiplier in the first column, which is $2$. This leaves a remainder of $3$, which is a multiple of $1$, so we should also note it.

Power of 8 ($x$) 1 0
$8^x$ 8 1
Fits in 2 3

Converting from Octal

To convert from octal to decimal, we can simply multiply each number by $8^x$ and add them together.

$$8 \times 2 + 1 \times 3 = 19$$

Hexadecimal

Hexadecimal are converted in the same way that octal numbers are converted. The only difference is that the base is $16$ instead of $8$.
